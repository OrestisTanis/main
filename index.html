Vicky Mamareli Photograhy - Portfolio Website

The portfolio website of the photographer Vicky Mamareli, featuring a photo gallery with lazy and progressive image loading. I've also made a second app, only for the photographer, that acts as a mini-cms giving her the ability to easily create/edit/delete website content.

Developed with React.js and Firebase.

AirSenSe - Air quality instruments e-shop and analytics service.

AirSenSe is the result of a collaborative effort about an air quality sensors e-shop that among others, provides a service which collects measurement data from installed sensors and displays it to the user via a data analytics dashboard. It is not yet deployed but you can take a look at the source code if you'd like.

Developed with Java, Spring Boot, MySQL and React.js using a RESTful architecture and JWTs.

The Green Leaf - Online Store

The Green Leaf is an Angular App I developed that uses Google's Firebase as its back-end. For demonstration purposes, all authenticated users have been granted administrator privileges, meaning that they have access to the admin section which lets them create new products or modify existing ones, as well as view the client order history.

Developed with Angular 8.

Revision - Revisit Your Knowledge

ReVision is an educational free application I created in collaboration with an ophthalmologist, designed to help medical professionals in cultivating their knowledge in ophthalmology, either for studying, exam preparation, or plain curiosity purposes. Overall, ReVision provides an entertaining and interactive question framework to reinforce both essential and advanced knowledge, necessary for practical application.

Made with Unity.

SpaceTime Fighter

This is my retrowave endless runner project. The general idea was to combine key game design patterns of the endless runner game genre, such as semi-random level generation, achievement system, and progressive difficulty together with 80's vaporwave aesthetics with modern elements, in order to create a nostalgic, immersive playing experience.

Made with Unity.
